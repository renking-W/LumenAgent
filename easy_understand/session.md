  数据流

  CLI / HTTP 请求
      ↓  session_id
  chat_service.py
      ↓
  context_assembly.py  ← 读取 session 历史 → 组装上下文
      ↓
  LLM 响应
      ↓
  chat_service.py  ← append_message(session_id, ...)
      ↓
  summary_service.py  ← 检查是否需压缩当前 session
      ↓
  sqlite_conversation.py  ← 最终落库
